REGISTRO EN LINEA

Cualquier persona que desea ser cliente de su entidad, podrá realizar el registro y la apertura de sus cuentas de forma remota, a través del modulo de identificación de clientes, este modulo solicita datos básicos, imágenes del documento de identidad con fotografía y fotografías del cliente para comparar con el documento, en este proceso existe una etapa mas de control, luego de este proceso se realiza la creación del cliente, en caso de no pasar las validaciones se establece una conexión con el cliente por medios de comunicación digitales.
